01 - HTML – jednoduchá stránka s odrážkovým seznamem

Kostra stránky

<!DOCTYPE html>
<html lang="cs">
<head>
    <meta charset="UTF-8">
    <title>Moje stránka</title>
</head>
<body>
    <h1>Nadpis</h1>
    <p>Krátký odstavec textu.</p>

    <h2>Odrážkový seznam</h2>
    <ul>
        <li>První položka</li>
        <li>Druhá položka</li>
        <li>Třetí položka</li>
    </ul>

    <h2>Číslovaný seznam</h2>
    <ol>
        <li>Krok jedna</li>
        <li>Krok dva</li>
    </ol>
</body>
</html>
<ul> = unordered list (odrážky)
<ol> = ordered list (čísla)
uvnitř obojího <li>
