15 - Basic – program s logickými podmínkami

Struktura IF

IF podmínka THEN
    ' co se stane když platí
ELSEIF jiná podmínka THEN
    ' jiný případ
ELSE
    ' jinak
END IF

Příklad: kladné × záporné × nula

DIM x AS INTEGER
INPUT "Zadej celé číslo: ", x

IF x > 0 THEN
    PRINT "Číslo je kladné."
ELSEIF x < 0 THEN
    PRINT "Číslo je záporné."
ELSE
    PRINT "Je to nula."
END IF

Příklad: prospěl / neprospěl

DIM znamka AS INTEGER
INPUT "Známka (1-5): ", znamka

IF znamka >= 1 AND znamka <= 4 THEN
    PRINT "Prospěl"
ELSE
    PRINT "Neprospěl"
END IF

Operátory

  • Porovnání: =, <>, <, >, <=, >=

  • Logické: AND, OR, NOT
