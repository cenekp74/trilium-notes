13 - Algoritmy a programovací jazyky

Algoritmus

  • přesný popis postupu, jak řešit úlohu pomocí konečné posloupnosti kroků

Vlastnosti algoritmu

  • konečnost – v konečném čase skončí
  • determinovanost – v každém kroku je jasné, co dělat
  • determinismus – stejný vstup → stejný výstup
  • hromadnost – řeší celou třídu úloh
  • rezultativnost – dává výsledek
  • efektivnost – řeší úlohu přiměřeně rychle

Zápis algoritmu

  • slovní popis
  • pseudokód
  • vývojový diagram (start, akce, rozhodování, konec)
  • programovací jazyk

Základní řídicí struktury

  • sekvence – kroky za sebou
  • větvení – if / else (podle podmínky)
  • cyklus
    • s pevným počtem opakování (for)
    • s podmínkou na začátku (while)
    • s podmínkou na konci (do-while, repeat)

Programovací jazyky – dělení

  • strojový kód – přímo procesoru, binární
  • nižší jazyky (assembler) – symbolický zápis instrukcí, závisí na procesoru
  • vyšší jazyky – nezávislé na hardware, blíž lidskému jazyku

Podle způsobu zpracování

  • kompilované – přeloží se celé do strojového kódu (C, C++, Pascal, Go, Rust)
  • interpretované – kód se vykonává řádek po řádku (Python, JavaScript, PHP)
  • hybridní – kompilace do mezikódu, pak interpretace (Java – JVM, C# – .NET)

Podle paradigmatu

  • procedurální – posloupnost procedur (C, Pascal)
  • objektově orientované (OOP) – třídy a objekty (Java, C++, C#, Python)
    • zapouzdření, dědičnost, polymorfismus, abstrakce
  • funkcionální – funkce jako základ (Haskell, Lisp, F#)
  • logické – pravidla a fakta (Prolog)
    • nejvic husty ass ale je to legit uplne k nicemu
  • skriptovací – pro krátké úlohy (Bash, JavaScript)

Příklady populárních jazyků

  • Python – univerzální, AI, data science, snadná syntaxe
  • JavaScript – web, frontend i backend (Node.js)
  • Java – enterprise, Android, multiplatformní
  • C / C++ – systémy, hry, vysoký výkon
  • C# – Microsoft, .NET, hry (Unity)
  • PHP – web (server-side)
  • SQL – dotazovací jazyk pro databáze
  • HTML, CSS – značkovací (ne programovací v užším smyslu)
  • Rust - jako C akorát lepší jupí a maj ve znáčku hustýho raka

Příklady algoritmů

  • třídicí: bubble sort, insertion sort, quicksort, merge sort
  • vyhledávací: lineární, binární
  • grafové: Dijkstra (nejkratší cesta), BFS, DFS
  • šifrovací: AES, RSA, hashe (SHA, MD5)
