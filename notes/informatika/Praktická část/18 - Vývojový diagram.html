18 - Vývojový diagram

Značky (musíš znát!)

  • Ovál ⬭ – začátek / konec programu (Start, Konec).

  • Rovnoběžník ▱ – vstup / výstup (Načti x, Vypiš y).

  • Obdélník ▭ – akce / zpracování / přiřazení (y = x + 1).

  • Kosočtverec ◇ – rozhodnutí (podmínka) – vede z něj ANO / NE.

  • Šipky → – směr toku.

  • Kruh ○ – spojka (pokračování na jiném místě).

Pravidla

  • Vždy jeden Start a jeden Konec.

  • Z akce vede jedna šipka pryč, z podmínky dvě (ANO/NE).

  • Šipky se nekříží (pokud možno).

  • Tok je shora dolů, zleva doprava.

Příklad – sudé/liché

│  Start  │
└────┬────┘
