14 - Software – přehled

Co je software

  • programové vybavení počítače – nehmotná část
    • nemůžu do toho kopnout
  • protiklad k hardwaru (fyzická část)
    • do toho můžu kopnout

Systémový software

  • operační systém (OS)
    • základní program, ovládá hardware a poskytuje služby aplikacím
    • úkoly: správa procesů, paměti, souborů, zařízení, uživatelů
    • windows, macOS, Linux (Ubuntu, Fedora, Debian…), Android, iOS
      • macOS, Linux, Android jsoiu všechno UNIXový systémy
  • ovladače (drivers) – umožňují OS komunikovat s hardwarem
  • firmware – software pevně spojený s hardware (BIOS, interní kód externích karet (ne jako želviček), disků, …)
  • systémové nástroje – defragmentace, čištění disku, správce úloh

Aplikační software

  • kancelářské balíky – MS Office, LibreOffice, Google Docs
  • webové prohlížeče – Chrome, Firefox, Edge, Safari, Opera, …
  • multimédia
    • přehrávače (VLC, Media Player)
    • editory obrazu (Photoshop, GIMP)
    • editory videa (Premiere, DaVinci, Shotcut)
    • editory zvuku (Audacity)
  • grafické programy – Illustrator, Inkscape, Blender
  • vývojové nástroje – Visual Studio, IntelliJ, VS Code, PyCharm
  • komunikace – Outlook, Thunderbird, Teams, Discord, Slack
  • databáze – MySQL, PostgreSQL, Oracle, MS SQL
  • antiviry – ESET, Avast, Kaspersky, Windows Defender
  • specializované – CAD (AutoCAD), účetnictví (Pohoda), DTP (InDesign)

Rozdělení podle licence

  • komerční (proprietární) – placený (Windows, Photoshop)
  • freeware – zdarma, ale ne s otevřeným kódem
  • shareware – zdarma na zkoušku
  • open source – otevřený zdrojový kód (Linux, Firefox, LibreOffice)
  • public domain – bez omezení
  • adware – zdarma s reklamou

Distribuce softwaru

  • krabicová verze (fyzické médium)
  • ke stažení z webu (digitální distribuce)
  • předplatné (SaaS – Software as a Service: Office 365, Adobe CC)
  • obchody s aplikacemi (Microsoft Store, App Store, Google Play)

Životní cyklus softwaru

  • analýza požadavků → návrh → implementace → testování → nasazení → údržba
  • Verze (1.0, 2.5.1), aktualizace, bezpečnostní záplaty
