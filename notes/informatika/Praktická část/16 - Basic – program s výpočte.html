16 - Basic – program s výpočtem

Obsah obdélníku

DIM a AS SINGLE, b AS SINGLE, obsah AS SINGLE

INPUT "Strana a: ", a
INPUT "Strana b: ", b

obsah = a * b
PRINT "Obsah obdélníku je: "; obsah

Průměr ze tří čísel

DIM x AS SINGLE, y AS SINGLE, z AS SINGLE, prumer AS SINGLE

INPUT "Číslo 1: ", x
INPUT "Číslo 2: ", y
INPUT "Číslo 3: ", z

prumer = (x + y + z) / 3
PRINT "Průměr je: "; prumer

Převod Kč na € (s kurzem)

DIM kc AS SINGLE, kurz AS SINGLE, eur AS SINGLE
kurz = 24.5
INPUT "Částka v Kč: ", kc
eur = kc / kurz
PRINT kc; " Kč = "; eur; " EUR"

Datové typy

  • INTEGER – celé číslo

  • SINGLE / DOUBLE – desetinné

  • STRING – text (např. jmeno AS STRING)

Aritmetické operátory

+ - * / ^ (mocnina) MOD (zbytek) \ (celočíselné dělení)
