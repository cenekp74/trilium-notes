03 - CSS – HTML stránky s kaskádovými styly

Tři způsoby zápisu CSS

inline: <p style="color: red;">…</p>

interní - v <head> uvnitř<style>…</style>

externí

Externí styl – propojení

<head>
    <link rel="stylesheet" href="styl.css">
</head>

Obsah souboru styl.css

body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    margin: 20px;
}

h1 {
    color: #2c8fbd;
    text-align: center;
}

.zvyrazneni {       /* třída – v HTML: class="zvyrazneni" */
    background: yellow;
    font-weight: bold;
}

#hlavni {           /* id – v HTML: id="hlavni" */
    border: 2px solid black;
    padding: 10px;
}

p {
    line-height: 1.5;
    color: #333;
}

Selektory

  • tag – platí na všechny tagy (např. všechny <p>)

  • .trida – platí na class="trida"

  • #id – platí na id="id" (id musí být unikátní)

Časté vlastnosticolor, background-color, font-size, font-family, text-align, margin, padding, border, width, height.
